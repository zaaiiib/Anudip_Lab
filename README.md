# Anudip MySQL Lab Practice 🚀

Welcome to my official repository for all MySQL practical exercises, lab assignments, and SQL command practice completed during my course at **Anudip Foundation**.

This repository serves as a personal log of my journey learning relational database management systems, moving from basic database creation to advanced querying.

---

## 📂 What This Repository Contains

Here is a summary of the SQL concepts and commands practiced and documented here:

* **DDL (Data Definition Language):** Creating, altering, dropping, and truncating databases and tables (`CREATE`, `ALTER`, `DROP`, `TRUNCATE`).
* **DML (Data Manipulation Language):** Inserting, updating, and deleting records (`INSERT`, `UPDATE`, `DELETE`).
* **DQL (Data Query Language):** Retrieving data using the `SELECT` statement and filtering results.
* **Pattern Matching & Filtering:** Using the `LIKE` operator along with wildcards (`%` and `_`) for precise text searches.
* **Constraints:** Working with primary keys (`PRIMARY KEY`), `NOT NULL`, and data types (`INT`, `VARCHAR`).
